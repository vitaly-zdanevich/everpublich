Everpublich

Sync an Evernote notebook to a static Zola blog/website with RSS, tags, search, a calendar, podcast feed, media playback, expanded links-to-widgets, and nginx hosting on an ARM VM or on your GitHub.

A better version of postach.io and notesrss.com.

Your site will use a subdomain under __BASE_DOMAIN__

Preparing OAuth...

What ships in the MVP

  • Daily full regeneration
  • Zola static search enabled by default, with Google search as a setting
  • Images, audio, video, and attachments copied from Evernote and served by nginx on the ARM VM
  • RSS, sitemap.xml, tag pages, About page, calendar, and podcast.xml for notes tagged podcast
  • Optional GitHub backup repository with public/private visibility
  • Mobile-friendly design with true black dark mode via prefers-color-scheme
  • Offline support - right in a browser
