Everpublich

Sync an Evernote notebook to a static Zola blog/website with RSS, tags, search, a calendar, podcast feed, media playback, expanded links-to-widgets, and S3/CloudFront hosting or on your GitHub.

A better version of postach.io and notesrss.com.

Share your notebook read-only with everpublich@proton.me, and we will publish and sync it.

Your address will be https://d2ieo3xczytvos.cloudfront.net/<your-notebook-name>

Evernote no longer exposes native note tags in shared notebooks, so Everpublich reads #tags from note content instead.

What ships in the MVP

  • Daily full regeneration
  • Zola static search enabled by default, with Google search as a setting
  • Images, audio, video, and attachments copied from Evernote and served from S3 through CloudFront
  • RSS, sitemap.xml, tag pages from note-content #tags, About page, calendar, and podcast.xml for notes with #podcast
  • Optional GitHub backup repository with public/private visibility
  • Mobile-friendly design with true black dark mode via prefers-color-scheme
  • Offline support - right in a browser

Plans, if we have users

  • Nice subdomain address and custom domains
  • Sync to WordPress
